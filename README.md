# Jarvis AI Assistant

**Voice-first AI chief of staff for solo entrepreneurs on iPhone + Mac**

---

## Vision

Jarvis is a personal operating system for decisions and execution. A voice-first, cross-device AI assistant that runs your day, protects your focus, and executes work across your tools.

The test of success: **you stop being the router for everything. Jarvis becomes the router.**

## Status

**Factory Version:** v20.0 (Autonomous Mode)
**Project Stage:** Planning (Pre-execution)

## What You're Buying

- Fewer context switches
- Fewer dropped balls
- Faster throughput on repetitive work
- Better decision hygiene

## Core Capabilities (MVP)

1. **Unified Inbox** - Triage across email, messages, tasks
2. **Communications Operator** - Draft, rewrite, schedule followups
3. **Calendar Co-pilot** - Smart scheduling with pre-briefs
4. **Daily Planning** - "What matters today?" with top 3-5 outcomes
5. **Focus Protection** - Smart DND with batched notifications

## Tech Stack (Planned)

- **Platform:** iOS + macOS (SwiftUI)
- **Voice:** Native speech recognition + synthesis
- **Backend:** TBD (Convex/Supabase evaluation needed)
- **AI:** Claude API for intelligence layer
- **Integrations:** Apple Calendar, Mail, Reminders, Gmail
