# Signal Snapshot

A signal snapshot is a point-in-time summary of key signals.

Snapshots are used as inputs to the decision engine.

---

## Snapshot metadata
- date
- product
- generated_by (script / manual)
- data sources used

---

## Signals included
For each signal:
- name
- value
- trend
- confidence
- notes

---

## Interpretation
Short summary of what the snapshot suggests.

Snapshots do not mandate actions.
