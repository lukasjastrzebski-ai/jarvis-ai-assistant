# Product Documentation

This directory contains product definition artifacts created during **Stages 0-2** of the Ideation Playbook.

## Expected Files

| File | Stage | Purpose |
|------|-------|---------|
| `idea_intake.md` | Stage 0 | Raw idea capture without solution bias |
| `vision.md` | Stage 1 | Why this product should exist |
| `strategy.md` | Stage 1 | How success will be achieved |
| `metrics.md` | Stage 1 | How success is measured |
| `risks.md` | Stage 1 | Known risks and mitigations |
| `definition.md` | Stage 2 | What the product is and is not |
| `personas.md` | Stage 2 | Target user definitions |
| `journeys.md` | Stage 2 | Primary user journeys |
| `non_goals.md` | Stage 2 | Explicit exclusions from scope |

## Usage

1. Start with `idea_intake.md` during Stage 0
2. Complete Stage 1 documents before proceeding to Stage 2
3. All documents must exist before Stage 3 (Feature Discovery)

## Reference

See [docs/ideation_playbook.md](../ideation_playbook.md) for detailed guidance on each stage.
