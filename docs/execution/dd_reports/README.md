# DD Reports Directory

**Version:** 20.0

This directory contains reports generated by the Product Owner for the Delivery Director.

---

## Report Types

### Phase Completion Reports

Generated at the end of each phase.

**Filename:** `PHASE-XX-report.md`

**Contents:**
- Phase summary
- Tasks completed
- Tasks blocked/skipped
- Issues encountered
- Recommendations

### Daily Summaries

Generated daily during execution.

**Filename:** `daily_YYYY-MM-DD.md`

**Contents:**
- Day's progress
- Active work
- Blockers
- Next day plan

### Escalation Reports

Generated when escalations are resolved.

**Filename:** `escalation_ESC-XXX-resolved.md`

**Contents:**
- Escalation details
- Resolution
- Impact on execution

### Milestone Reports

Generated at major milestones.

**Filename:** `milestone_DESCRIPTION.md`

**Contents:**
- Milestone achieved
- Metrics
- Comparison to plan

|------|--------|-------|

### Blocked: Z

2. [Issue 2]

- Duration: X hours
- Agent hours: Y

## Report Generation

Reports are generated automatically by PO at:

- **Daily:** End of each day (or session)
- **Phase end:** When all phase tasks complete
- **Escalation resolution:** When DD responds to escalation
- **On request:** When DD requests STATUS or summary

---

## DD Access

DD can request reports via:

```
STATUS                  - Current status summary
DETAIL TASK-XXX        - Specific task details
ESCALATIONS            - Pending escalations

Or by reading files in this directory.
```
