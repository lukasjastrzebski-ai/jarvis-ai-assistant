# Skill 09 – CR / New Feature Router

Purpose:
Route scope changes to the correct gated flow.

Steps:
1) Detect scope change or missing requirement.
2) Classify:
   - behavior change → Change Request
   - net-new functionality → New Feature
3) STOP execution.
4) Point to correct flow document.
5) List required artifacts.

Output:
- routing decision
- next required documents
